Interface and Application programming

On this page i have recorded my process of programming my "Hello World" PCB

Using the Hello world board which was produced during the Electronics production, I wrote codes in both arduino and python which allowed the board to execute actions on my laptop

Arduino

In the arduino code, it was used to detect the button press from the board, send a serial output through the serial pins and turn on the LED

Python

The python code on the other hand takes the serial output from the arduino program and executes any desired code according to the outputted serial value
